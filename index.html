HTML & CSS Cheat Sheet

by Holy150

In this page you can find some of the most popular resources in HTML & CSS

Coding Resources

Resource Description
W3Schools One of the most popular and well-known resources for learning HTML and other web development technologies. It offers a wide range of tutorials and examples, as well as interactive quizzes and a certification program. It's a great resource for beginners as well as for more advanced developers.
Mozilla Developer Network (MDN) Offers a wealth of information and resources on web development, including a comprehensive guide to HTML. It's a great resource for learning about web standards and best practices, and it's maintained by a community of experts.
Codecademy Offers a comprehensive, interactive HTML course that includes quizzes, projects, and practice exercises. It's a great resource for beginners and it also offers courses for other programming languages and web development technologies.
HTML Dog A website that provides HTML tutorials, articles and examples. The website has a clean and simple design, and it's great for finding specific HTML topics and learn more about it.
FreeCodeCamp A non-profit organization that offers a comprehensive curriculum of coding lessons, including a section dedicated to HTML. It's a great resource for beginners as well as for more advanced developers, and it also offers a community forum where you can ask questions and get help from other developers.

CSS Resources

Resources Description
CSS-Tricks This website offers a wide range of tutorials and articles on CSS, as well as a blog with the latest news and tips on web development. It's a great resource for learning about advanced CSS techniques and best practices, and it's maintained by a community of experts.
CSS Zen Garden This website is a showcase of the design capabilities of CSS. It's a great resource for learning about CSS layout and design, and it also offers a gallery of designs submitted by other designers. You can look at the source code of each design to learn different techniques.
A List Apart This website is dedicated to web design and development, and it offers a wide range of tutorials, articles, and resources on a variety of topics, including CSS. It's a great resource for learning about web standards and best practices.
Can I Use This website provides a compatibility table that shows which features are supported by different browsers. It's a great resource for checking browser support for different CSS properties and features, so you can make sure your website is compatible with different browsers.
Sass-lang Sass is a preprocessor scripting language that is interpreted or compiled into Cascading Style Sheets (CSS). Sass adds features such as variables, nested rules, mixins, and functions, and it helps to keep large stylesheets well-organized. The official website of Sass is a great resource for learning the language and its features and best practices.
